## 🌍 서비스 기획 배경

여행은 함께할 때 더 즐겁습니다. 
<br>그러나 **예산과 일정이 서로 다른 유럽 여행자**들은 홀로 여행하는 경우가 많아, **외로움과 안전에 대한 불안감**을 느낍니다.
<br>**Urabi**는 이러한 여행자들이 **장기 동행과 번개 모임**을 통해 **믿을 수 있는 동행자를 찾고**, **숙소 후기**와 **여행 물품**을 공유하며 
<br>**더 안전하고 편리한 여행**을 즐길 수 있도록 돕는 **지도 기반 여행 동행 & 커뮤니티 플랫폼**입니다&#8203;

### 기획 의도
- 유럽 여행자들의 **외로움과 안전 문제**를 해결하고, **비슷한 여행 스타일을 가진 사람들과 연결**될 수 있도록 돕는 서비스입니다.  
- **장기 동행**뿐만 아니라, **즉흥적인 번개 모임** 기능을 통해 짧은 시간 함께할 여행자를 찾을 수 있습니다.  
- **한국인의 믿을 수 있는 숙소 후기**와 **여행 중 필요 없는 물품 교환·판매 기능**을 제공하여,
  <br>유럽 여행 중 발생할 수 있는 여러 **불편함**을 해결하고자 했습니다.  
- **지도를 기반으로 한 커뮤니티 결합**이라는 차별점으로, **직관적이고 효율적인 여행자 연결**을 목표로 합니다.

### Urabi의 특징  
- **신뢰성과 안전성**: **카카오·네이버 소셜 로그인**, **본인 인증**, **사용자 평가(매너 온도)**, **신고 관리 시스템**을 통해
  <br>안심할 수 있는 여행 환경을 제공합니다.  
- **실시간 채팅 지원**: 웹사이트 내 **실시간 채팅 기능**을 구현하여, 카카오톡·네이버 카페 등 외부 플랫폼에 의존하지 않고
  <br>**사이트 내에서 모든 커뮤니케이션**이 가능합니다.  
- **독립적이고 체계적인 시스템**: 기존 대규모 커뮤니티(예: 유랑)가 네이버 카페 시스템에 의존하는 한계를 극복하고,
  <br>**Urabi만의 독립적인 플랫폼**에서 **동행 매칭부터 후기, 거래까지** 제공하는 것이 차별점입니다.

**Urabi**에서 동행을 찾아, 보다 **안전하고 잊지 못할 유럽 여행**을 만들어보세요!

## 🔗 서비스 매커니즘 & 사용 가이드

## 🗺️ 여행 동행 찾기
### 👤 동행 모집자
1. 여행 계획을 등록하거나, 지도에서 여행 지역을 선택하여 동행 모집 게시글을 작성합니다.  
2. 여행 일정, 나이·성별 등의 필터를 설정하고 태그로 관심사를 설정한 후 모집글을 게시합니다.  
3. 모집글을 본 사용자가 채팅으로 문의를 보내면, 모집자가 확인 후 승인합니다.  
4. 승인된 사용자는 동행자로 추가되며, 실시간 채팅으로 여행 일정을 조율할 수 있습니다.

### 🤝 동행 참여자
1. 지도에서 동행 모집 게시글을 검색하고, 날짜·지역·스타일(태그기반) 등을 필터링하여 확인합니다.  
2. 관심 있는 모집글에 실시간 채팅을 보냅니다.  
3. 모집자가 요청을 승인하면, 동행자로 추가되어 채팅으로 소통하며 여행을 준비합니다.

---

## ⚡ 여행 번개 (즉흥 여행 모임)
### 📝 번개 모임 등록자
1. 지도에서 번개 모임 위치를 선택하고, 제목, 날짜, 시간, 인원, 태그 등을 설정하여 게시글을 등록합니다.  
2. 디테일 페이지에서 +버튼을 눌러 참여 요청을 보낸 사용자를 확인하고, 실시간 채팅을 통해 소통한 후 참여 여부를 승인합니다.  
3. 승인된 사용자는 번개 참여자 목록에 추가되며 디테일 페이지에서, 필요 시 채팅으로 모임 세부사항을 논의할 수 있습니다.

### 🚀 번개 모임 참여자
1. 지도에서 번개 모집 게시글을 검색하고, 날짜·장소·모임 스타일을 확인합니다.  
2. 관심 있는 모임에 참여 요청을 보냅니다.  
3. 모임장의 승인을 받으면 참여자로 추가되어, 채팅으로 모임 세부사항을 공유하고 모임 당일 함께할 준비를 합니다.

---

## 🏡 숙소 후기 공유
### 🖊️ 숙소 후기 작성자
1. 지도에서 숙소를 검색한 뒤 자신이 머문 숙소 정보를 입력하고, 평점과 함께 후기를 작성합니다.  
2. 작성된 후기는 지도 기반으로 노출되어 다른 사용자가 확인할 수 있습니다.

### 🔍 숙소 후기 열람자
1. 지도에서 숙소 위치를 선택하고, 필터(위치, 평점)를 적용해 필요한 숙소 후기를 검색합니다.  
2. 원하는 숙소를 찾으면 세부 후기를 열람하고, 다른 여행자들의 평가 및 사진을 참고해 선택합니다.  
3. 열람자도 댓글 기능을 통해 세부 후기를 추가할 수 있고, 도움이 된 후기에 좋아요를 달아줄 수 있습니다.

---

## 🎁 나눔 마켓
### 🛒 거래 등록자
1. 판매할 물품을 등록하고, 카테고리(의약품, 티켓, 옷, 생활용품 등)를 설정해 게시글을 작성합니다.  
2. 실시간 채팅으로 거래 요청을 받고, 거래 여부를 결정합니다.  
3. 거래 완료 이후 거래 가능 여부를 갱신합니다.

### 🔄 거래 참여자
1. 장소나 카테고리 검색을 통해 필요한 물품을 찾고, 게시글을 열람합니다.  
2. 실시간 채팅으로 거래 요청을 보내고, 거래 등록자와 협의하여 거래를 진행합니다.



## 💡 페이지별 개요

#### 🏝 장기 동행

- 지도를 활용해 생성한 여행 일정을 기반으로 동행을 찾을 수 있음
- 사용자가 방문하는 여행지를 기반으로 동행을 찾을 수 있음
- 필터를 적용하여 비슷한 여행 스타일을 가진 동행 검색 가능
- 동행 모집 글 작성 및 수정 기능
- 참여하기 및 실시간 채팅 기능 제공

#### 🔥 번개 모임

- 즉흥적인 여행을 위한 번개 모집 기능
- **불특정한 장소**에서 만날 수 있는 번개의 특성을 적용해서 **스트리트 뷰**를 도입
- 필터를 적용하여 원하는 스타일의 번개 모임 검색 가능
- 사용자의 위치를 활용하여 가까운 번개 조회 기능
- 참여하기 및 실시간 채팅 기능 제공

#### 🏡 숙소 후기

- 여행자들이 머문 숙소의 정보를 공유
- 위치, 별점 등의 정보를 포함한 리뷰 작성 가능
- 검색 및 필터링 기능을 통해 원하는 숙소 정보 검색 가능

#### 🎁 나눔 마켓

- 여행 중 필요한 물품 혹은 불필요한 물품을 여행자들끼리 거래 가능
- 사용자가 올린 물품에 대해 실시간 문의 및 채팅 가능
- 두 명이서 합의하여 장소를 정하는 경우가 많기에 지도를 도입하기보다는 **도시 기반 검색**으로 설계
- 
#### 🧑‍💼 마이페이지

- 회원 정보 수정 기능 제공
- 나의 여행 일정 관리 페이지에서  장기 동행 일정 등록
- 작성한 동행 모집 글, 번개 모임, 숙소 후기, 나눔 마켓 내역 확인 가능
- 찜한 동행 모집 글, 번개 모임, 숙소 후기, 나눔 마켓 내역 확인 가능

#### 🤖 챗봇 페이지

- 여행 설계에 도움을 줄 수 있는 간단한 챗봇
- **GPT 3.5** 사용

#### 💬 채팅 페이지

- 1:1 **실시간** 채팅 기능 제공
- 모든 장기 동행, 번개, 나눔 마켓은 채팅을 통해 연결
- 이름 기반 검색은 충돌의 문제가 있어 고유한 이메일로 사용자를 검색하여 사용자끼리 소통할 수 있게 함
- 새로운 메시지 도착 시 **알림 기능** 제공
- **웹소켓**을 이용하여 구현
